# RemitZim

RemitZim is a single-page application for a remittance service that allows users to send money to Zimbabwe and other African countries. The application is built with vanilla JavaScript, HTML, and CSS, and it uses Tailwind CSS for styling.

## Features

- User authentication (sign in, sign up)
- View transaction history
- Send money to beneficiaries
- Add and manage payment methods
- Top up wallet
- Responsive UI for mobile and desktop devices

## Technologies Used

- HTML
- CSS
- JavaScript
- Tailwind CSS

## Getting Started

To get started with the project, clone the repository and open the `index.html` file in your browser.

```bash
git clone https://github.com/your-username/remitzim.git
cd remitzim
```
